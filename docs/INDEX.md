## Convention de nommage

- Fichiers maîtres : `MAJUSCULES_AVEC_UNDERSCORES.md` (ex. `ARCHITECTURE.md`, `CHANGELOG.md`)
- Fichiers métier : `Title_Case_With_Underscores.md` (ex. `Pitch_Deck_10_slides.md`)
- Sous-dossiers thématiques : `kebab-case` ou `PascalCase` (ex. `Enquete_Terrain/`, `security/`, `video_explanations/`)
- Pas d'emojis dans la documentation technique (sauf README principal pour lisibilité).
