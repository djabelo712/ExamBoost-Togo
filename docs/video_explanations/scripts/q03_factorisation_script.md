## Script voix off (FR)

> "Pour factoriser x² moins 9, on reconnaît une différence de deux carrés. Identité remarquable : a² moins b² égale a moins b fois a plus b. Ici, a égale x et b égale 3, parce que 9 égale 3². Donc x² moins 9 égale x moins 3 fois x plus 3. Voilà, c'est factorisé !"

(60 mots, environ 24 secondes à 130 mots/min)

## Script voix off (EN) — version internationale

> "To factor x squared minus 9, we recognize a difference of two squares. The identity : a squared minus b squared equals a minus b times a plus b. Here, a is x and b is 3, because 9 is 3 squared. So x squared minus 9 equals x minus 3 times x plus 3. There you go, factored !"

(58 words, about 24 seconds at 130 wpm)

## Notes de prononciation et d'intonation (FR)
- "x² moins 9" : prononcer "ix carré moins neuf", marquer une pause après.
- "a² moins b²" : bien articuler "a carré", "b carré".
- "égale a moins b fois a plus b" : décomposer les opérations, pas de liaison entre "a" et "moins".
- "9 égale 3²" : insister sur le "3 au carré", faire le lien avec le 9 du départ.
- "Voilà, c'est factorisé !" : ton triomphant, conclusion claire.

## Astuces pédagogiques
- Bien faire le lien "9 = 3²" — c'est le déclic pour les élèves : ils ne voient pas toujours que 9 est un carré parfait.
- Mentionner les 3 identités remarquables à connaître par cœur :
  - (a + b)² = a² + 2ab + b²
  - (a - b)² = a² - 2ab + b²
  - a² - b² = (a - b)(a + b)
- La 3e (différence de deux carrés) est la seule qui sert à FACTORISER ; les deux autres servent à DÉVELOPPER. C'est un point clé souvent confondu.
- Variante "astuce examen" : si l'élève bloque, vérifier que chaque terme est un carré parfait. Sinon, la factorisation est différente (par exemple x² - 6x + 9 = (x - 3)²).
